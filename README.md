# Face-Recognition
Usage : Enter name and number of people on the screen 
Once that is done you should see a camera feed of your image 
Then press 's' to start taking traing data of your face and generate fisher faces dynamically through the code 
that is it ! You are done . You should be able to see your face recognised at sandard 33 fps . 

Possible errors :
Please make sure that you have installed all dependencies .
This was attempted on windows. Please make sure that the pathway that the code is generating seems apt for linux or mac users as well
